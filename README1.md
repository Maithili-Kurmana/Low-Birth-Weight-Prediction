# Fetal Birth Weight Prediction

This is a web-based application built using Flask and Machine Learning.

## Features
- User Registration and Login
- Predict fetal birth weight
- Uses Random Forest Algorithm

## Technologies Used
- Python
- Flask
- MySQL
- HTML/CSS
- Machine Learning

## How to Run
1. Install required libraries
2. Run app.py
3. Open http://127.0.0.1:5000

## Output
- Predicts Low, Normal, or Abnormal Birth Weight
